# Templates ZPL

Esta pasta contém templates ZPL (Zebra Programming Language) para geração de etiquetas.

## Templates Disponíveis

- **LAYOUT_ETIQUETA_FINAL.zpl** - Layout final da etiqueta
- **LAYOUT_FINAL_FUNCIONAL.zpl** - Layout funcional final
- **modelo_zpl_larroude.prn** - Modelo ZPL para Larroudé
- **ZPL_PYTHON_FUNCIONA.zpl** - Template que funciona com Python
- **ZPL_SISTEMA_PROBLEMA.zpl** - Template com problemas conhecidos
- **ZDesigner ZD230-203dpi ZPL** - Template para ZD230
- **ZDesigner ZD621R-203dpi ZPL** - Template para ZD621R
